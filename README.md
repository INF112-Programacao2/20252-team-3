Diário Alimentar

O sistema analisa a ingestão diária de alimentos do usuário e fornece um resumo detalhado de calorias e macronutrientes, baseando-se em algum banco de dados (TACO - Tabela Brasileira de Composicao de Alimento) para encontrar essas informações.

Execução do programa

Para executar o programa :
1) Abrir a branch "branch-limpa"
2) Selecionar a opção "Code"
3) Instalar o zip file com todos os arquivos da branch
4) Executar "make" no terminal para gerar o executável 

Funcionamento do programa 

1 -- Criando diretorio para salvar os seus dados

A primeira etapa do programa é informar o caminho de uma pasta onde ficarão salvos todos os seus dados que informar para programa (nome, idade, sexo, condicionamento fisico), os alimentos que você consumiu e os nutrientes desses alimentos. Nessa etapa você deve passar um caminho de diretorio onde ficará guardado todos os arquivos, recomenda-se ser o mesmo diretorio com os arquivos do programa.

2 -- Cadastro do Usuario

Aqui o usuario deve informar dados pessoais que serão utilizados para calcular a meta diária de calorias, carboidratos, proteinas, entre outros nutrientes, além de informar seu condicionamento fisico (sedentario, ativo), informação que é essencial para garantir que o calculo das metas nutricionais estejam de acordo com o estilo de vida.

3 -- Menu Principal

Nesse momento o usuario tera diversas opcoes de funcionalidades do programa para escolher, mas a principal delas, que é o objetivo central do programa, é a opcao de "Abrir o Diario Alimentar"

4 -- Diario Alimentar

Ao abrir a opção "Diario Alimentar", o usuario poderá adicionar novos alimentos ao seus diário, editar ou remover alimentos já adicionados e visualizar os alimentos e seus nutrientes que foram adicionados ao programa. A funcionalidade de adicionar o alimento busca no banco de dados por alimentos com nomes parecidos e mostra ao usuario essas opcoes de alimentos, o usuario escolhe uma delas e esse alimento é entao salvo em seu diario e junto ao alimento vem todos os nutrientes que os compoem, seja os Macronutriente (Calorias, Carboidrato, Lipidio, Proteina), seja os Micronutrientes (Vitaminas e Minerais)
